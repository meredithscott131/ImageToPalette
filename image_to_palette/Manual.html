Image To Palette

A docker that allows you to quickly generate color palettes from images.

Basic Usage

Go to Settings → Dockers and enable Image to Palette.

When the docker is open, navigate to the top-left icons Load Image and Regenerate Palette. Click Load Image to select the desired image you want to convert to a color palette. Once selected, you will see a 15-color grid of the most dominent colors from the image. If you aren't satisfied with the selection, click the Regenerate Palette button to select a new set of colors. To generate a palette from a different image, simply re-click the Load Image button.

Happy generating!
