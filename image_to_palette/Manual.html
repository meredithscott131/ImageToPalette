Image To Palette

A docker for quickly generating color palettes from images. Allows for saving palettes as JSON files.

Basic Usage

Go to Settings → Dockers and enable Image to Palette.

Features:

  • Create Palette From Image - Button to load a new palette from an image file.
  • Load Palette - Button to load a palette JSON file.
  • Save Palette - Button to save the current palette data as a JSON file.
  • Recent Palette History - Drop-down that lists the 5 most recently opened palette JSON files to select from.
  • Regenerate Palette - Button that regenerates the color palette from the current loaded image.
  • Previous/Next - Buttons for toggling between previously regenerated palettes from the same image.

Happy generating!
